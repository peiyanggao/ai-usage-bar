# Screenshots

Drop these PNGs here and the images in the main README will render automatically:

| File | What to capture | Suggested width |
| --- | --- | --- |
| `menubar.png` | The status item in the menu bar (`Claude NN%` + battery icon). Crop tight around it. | ~340 px |
| `dropdown.png` | The open dropdown with the two-row segmented bars (5-hour / weekly). | ~340 px |
| `touchbar.png` | The Touch Bar strip showing `Claude Code \| Codex`. | ~680 px |

## How to capture

- **Menu bar / dropdown:** press `⇧⌘4`, then `Space` to capture a single window/region,
  or drag a tight selection. Files land on your Desktop — rename and move them here.
- **Touch Bar:**
  - On a Mac **with** a Touch Bar: `⇧⌘6` saves the Touch Bar to your Desktop.
  - On a Mac **without** one (e.g. Apple Silicon): open the app in Xcode and use
    **Window ▸ Touch Bar ▸ Show Touch Bar** (the simulator), then `⇧⌘6` or screenshot it.

## Tips

- Use a clean desktop / a single theme (light or dark) for consistency.
- Keep the three images visually consistent (same appearance, similar scale).
- PNG with transparent background looks best for the menu-bar crop.
